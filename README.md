## Contributing

We welcome contributions! To add yourself or nominate someone to this list, please ensure the following format:

- **Name** - Full name
- **Position (+ years)** - Current position and years of experience
- **Skills (up to 10)** - Technical skills and expertise areas
- **CV/Resume** - Link to CV/Resume
- **LinkedIn** - Professional LinkedIn profile
- **GitHub** - GitHub profile showcasing code
- **Twitter** - Twitter/X profile (optional)
- **Portfolio** - Personal website or portfolio
- **NPM** - NPM profile (for JavaScript developers)
